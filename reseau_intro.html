Introduction aux Réseaux

Objectif du cours :

  • Assimiler les Bases des Réseaux

  • Savoir construire et administrer des Réseaux

Définitions :

Qu'est-ce qu'un Réseau ?

Dans sont sens le plus général, un réseau est une connection entre plusieurs entités, connection dont le but est l'échange d'information. Pour échanger des informations, ces entités vont devoir suivre des protocols

Qu'est-ce qu'un protocole ?

Un protocole est un ensemble de règles définissant la manière dont vont communiquer les entités. Dans les réseau ces protocoles sont variés et ont des rôle divers. Dans le cas des réseaux informatique on utilisera généralement un modèle en couche

Le modèle OSI

Dans le modèle théorique OSI chaque couche va produire un message qui sera encapsulé pour pouvoir être transmis à la couche inférieure

Avec ce modèle, la couche n d'une machine va pouvoir communiquer avec la couche n d'une autre machine

Par exemple la couche réseau d'une machine communiquera avec la couche réseau d'une autre machine

Les septs Couches du modèle OSI

  1. Application : Accès aux ressource

  2. Présentation : Conversion et chiffrement de données

  3. Session : Communication interhost

  4. Transport : Connection de bout en bout

  5. Réseau : Détermine le parcours et l'adressage logique

  6. Liaison : Adressage physique (Mac et LLC)

  7. Physique : Transmission Binaire en Numérique ou Analogique

La couche Physique :

Elle permet la transmission entre deux machines (transmission point à point) d'une suite de Bits. Cette couches peut avoir des interférences

La couche Liaison :

Cette couche assure la transmission de trame entre deux noeuds sur le même réseau local. C'est également une transmission de point à point, mais elle permet la correction des erreurs

La couche Réseau :

Elle se charge de la transmission des données entre deux noeuds, en passant éventuellement par des intermédiaire
