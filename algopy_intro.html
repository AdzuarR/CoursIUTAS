Quelques Définitions

L'Ordinateur :

L'ordinateur est capable de stocker les données en mémoire. De rechercher un données particulière en mémoire. D'écrire à l'écran. De récupérer des données tapées par l'utilisateur. D'effectuer des calculs sur les données

L'ordinateur effectue ces opérations à l'aide de programme, et pour comprendre ces programme nous allons devoir avoir une représentation de la machine. Pour cela on s'intéressera au contenu de la mémoire, et à ce qui est affiché à l'écran

Les Variables et leur Types

Les Variables

Elles vont nous servir à stocker les données, ce sont en fait des cases mémoire. En python les variables auront un Nom, une Valeur et un Type. On peut se les représenter mentalement comme des boites

Les types de Variables

En python on déclare une variable comme suit :

var = valeur
nombre = 5
nombre_virgue = 4.0
chaine_de_caractere = "Bob Marley"
Boolean = True
nombre_clone = nombre

Le signe "=" représente ici l'opération d'affectation, il permet d'affecter au nom de la variable a gauche, la valeur a droite. Ici nombre est le nom d'une variable qui prend 5 en valeur ce qui en fait une variable de type Entier, Int en Anglais. nombre_virgue prend 4.0 en valeur et est par conséquent un nombre à virgule, Float en Anglais. chaine_de_caractere prend "Bob Marley" en valeur, c'est un String (Signifie chaîne en Anglais). Boolean prend True, c'est un booléen, ces variables là ne peuvent prendre que deux valeur, True ou False.

Enfin nombre_clone est une variable qui prendra la valeur de la variable nombre, et sera donc 5 de type entier

Notion d'Environnement

Pour décrire l'état de nos programme on utilisera la notion d'environnement. L'environnement d'un programme constitue l'état de chacun des couple Variable/Valeur à un instant T du programme. Cette notion va permettre de comprendre finement comment fonctionne le langage et les différentes structure que nous utiliseront pour manipuler les données
