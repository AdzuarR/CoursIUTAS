Introduction

Dans ce cours de base de données, nous verrons comment à partir des informations que traitent une entreprise nous pouvons construire une structure permettant le traitement informatisés de ces données

Quelques définitions

Système d'informations :

C'est l'ensemble des moyens et des méthodes se raportant au traitement de l'information :

  • Saisie / Collecte
  • Stockage
  • Traitement
  • Diffusion / Affichage

Les systèmes d'informations présentent des intérêts pour les entreprises, notamment en terme d'automatisation des tâches.

Attention! Il ne faut pas confondre système d'information et système informatique!

Système informatique :

C'est la science du traitement automatique de l'information

Base de données / SGBD :

Un SGBD et un Système de Gestion de Bases de Données, il permet la gestion de Données structurées entre elles... Cool \o/...

La méthode MERISE

MERISE : Méthode d'Études et de Réalisation Informatique pour les Systèmes d'Entreprise

La méthode MERISE va permettre de créer des bases de données en suivant une sucession d'étapes depuis le recencement des données utilisées par l'entreprises jusqu'à l'implémentation de la base de donnée.
Créer une base de donnée va nécessité des capacités d'analyse de la part de l'informaticien pour effectuer les étapes suivantes :

  1. Recencement des données / Création du dictionnaire de données
  2. Construction du MCD
  3. Transformation du MCD en MLD
  4. Implémentation de la base de donnée

Le dictionnaire de donnée
