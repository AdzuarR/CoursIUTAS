Cours de Bases de Données

Système d'information

Ensemble de moyens et de méthodes se raportant au traitement automatique de l'information. Dans les entreprises cela comprend 4 fonction principales :

  • Saisie et collecte des Données

  • Stockage des Données

  • Traitement des Données

  • Diffusion et Affichage des Données

Il ne faut pas confondre Système d'information et système informatique

L'analyste programmeur va s'intéresser aux système d'information des entreprises pour voir si :

  • Une automatisation est possible

  • Si une application peut être créée

  • L'analyse informatique va nous donner des modèle pour structurer notre future application

Base de données

C'est un ensemble de données structurées et liés entre elles par un SGBD

SGBD

C'est un Système de Gestion des Bases de Données, il permet le Stockage l'exploitation, la recherche et la maintenance des Bases de Données

Principes Préalables à la création de la Base de Donnée

Avant la construction de la base de donnée à proprement parlé, il est nécessaire de recueillir les informations importantes qui devront être gérées par la BDD. Le contexte de ces informations est essentiel et doit également être pris en compte

Recencer les informations. Dans un premier temps on va faire la liste des informations importantes auxquel on donnera le nom de "donnée", et on construira à partir de ces données le dictionnaire de données

Organiser les informations. Il va s'agir de chercher dans les informations que l'on possède, qu'elles vont être les dépendances fonctionnelles entres les données. C'est à dire, préciser si en connaissant une donnée, cela me donne des informations non ambigues sur une autre données

Faire le MCD. Cette étape permet d'avoir un schéma de notre Base de Données, il doit absolument contenir toute les Données inscrites dans le dictionnaire de données, et le dictionnaire doit également contenir toute les propriétés présentent dans le MCD
